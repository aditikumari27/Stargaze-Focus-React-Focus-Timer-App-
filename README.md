# Stargaze Focus

A small, interactive React app (Vite) for focused study sessions:
- Pomodoro-style timer (presets)
- Persistent task list (localStorage)
- Circular progress, animated starfield and responsive layout
- No account required

## Run locally
1. `npm install`
2. `npm run dev`
3. Open the dev URL (usually http://localhost:5173)

Enjoy and customize colors/presets to your liking!
